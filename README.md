## Por que disponibilizar o projeto?
Quis disponibilizar o projeto, pois apesar dos pesares, o sistema ficou com uma interface bonita e intuitiva para o usuário.<br/>
**Screenshots**: [link](https://drive.google.com/drive/folders/19TymEMrxDnPQT0Vfw1FNBfW1iUs16bUM?usp=sharing)

É um sistema desktop desenvolvido utilizando **JavaFX, PostgreSQL e Jasper (relatórios)**.

**Algumas características**:
- Sistema para gerenciar condomínios com um controle total das atividades;
- No primeiro acesso, permite configurar a conexão com o banco de dados;
- No primeiro acesso, permite cadastrar e definir um usuário que será administrador do sistema;
- Cadastro de pessoas, veículos, prestadores de serviços;
- Controle de movimentações de entrada/saída;
- Controle financeiro;
- Acompanhamento de eventos (festas, reuniões familiares, etc), tudo aquilo que gera movimentação de novas pessoas;
- Relatórios detalhados;
- Backup e restauração do banco de dados;
- Dentre outras.
